Setting up a static IP address

  • This is especially useful if you need to connect remotely. Most devices dynamically set a new IP after a reboot
  • Modify /etc/dhcpcd.conf, which is the DHCP client daemon. See References for more details
    • In short, we need the following information:
      1. The network, which is eth0 or wlan0
      2. The static IP, which we can get from hostname -I
      3. The router's gateway IP address from ip r | grep default
      4. The router's DNS IP address from sudo vim /etc/resolv.conf. The DNS address is usually the same as the gateway
  • Add these lines to dhcpcd.conf:
interface <network>
    static ip_address=<static ip>/24
    static routers=<router ip>
    static domain_name_servers=<dns ip>
  • 1: 192.168.1.26 and 2: 192.168.1.27 are what we're using
  • 3: 192.168.1.28 (the RPi model B+ named "pi")
    • or 128.113.122.188 on ethernet with rpi_wpa2
