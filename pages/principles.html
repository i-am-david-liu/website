Digital garden principles, ordered by importance:

  1. Tend diligently. Trim, grow, and harvest; preferably daily. Treat this site like you're a real gardener. If you become careless, the plants will wither.
  2. Forget your audience. Never curate, never censor. The garden is an extension of the mind; therefore, all content should be sincere and genuine.
  3. Do it yourself. This website is built with no-frills HTML, CSS, and Javascript. Minimize the use of third-party tools and frameworks. If you want to add a feature or streamline the development process, make it from scratch and share what you learned.
  4. Scale downwards. This is not a place to enforce cutting-edge web development or perform database queries. This is a place for thoughtful notes and ideas. Always choose the yard over the greenhouse, and don't implement striking changes unless they're really, really, really cool.
